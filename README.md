# graphql-idors-sast

## Installation

- ```git clone https://github.com/grishxnder/graphql-idors-sast.git```
- ```cd graphql-idors-sast```
- ```You can make the venv (this is optional).```
- ```pip install -r requirements.txt```
  
Make the venv:
- ```python3 -m venv myapp```
- ```source myapp/bin/activate```
